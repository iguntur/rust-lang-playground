# Rust lang playground

> How to test [rust](https://www.rust-lang.org) private / internal function

---

## Guides

### Clone

```console
$ git clone https://github.com/iguntur/rust-lang-playground
$ cd rust-lang-playground
```

### Play with it

#### Compile and execute

```console
$ cargo run
```

#### Test

```console
$ cargo test
```
